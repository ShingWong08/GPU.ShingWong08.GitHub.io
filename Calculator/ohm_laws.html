歐姆定律及電功率

程式可以在已知歐姆定律及電功率公式其中兩個變(V、I、R及P)，計算其餘未知變數。

程式編寫日期: 2008年3月9日

程式 (80 bytes)

ClrMemory: ?→A: ?→B: ?→C: ?→D:

A=0 => BD => D÷B→A◢

A=0 => BC + √( CD→A◢

B=0 => C => A÷C→B◢

B=0 => D÷A→B◢

C=0 => A ÷B→C◢ AB→D

註1: 執行程式，輸入扇形變數的次序為電壓(V)、電流(I)、電阻(R)及電功率(P)，不輸入數值(必要) 代表未知數，即直接按EXE即可，而答案會依照輸入未知數時(直接按EXE)的次序顯示。

註2: 程式使用標準的SI單位，電壓單位為伏特(V)，電流單位為安培(A)，電阻單位為歐姆(Ω)，功率單位為瓦特(W)。

註3: 程式不能計算變數為0或負數的情況。

　

例題1: 某燈泡的電壓及電流分別為220V及0.25A，計算電阻及電功率。

按 Prog 1 再按 220 EXE 0.25 EXE

EXE EXE (不輸入電阻及電功率，顯示電阻為 880 Ω)

EXE (顯示電功率為 55 W)

　

例題2: 某燈泡的電壓及電阻分別為220V及880Ω，計算電流及電功率。

按 Prog 1 再按 220 EXE EXE (不輸入電流) 880 EXE

EXE (不輸入電功率，顯示電流為 0.25 A)

EXE (顯示電功率為 55 W)

　

例題3: 某燈泡的電壓及電功率分別為220V及55W，計算電流及電阻。

按 Prog 1 再按 220 EXE EXE EXE (不輸入電流及電阻)

55 EXE (顯示電流為 0.25 A)

EXE (顯示電阻為 880 Ω)

計算完結後按 AC 終止程式

　

例題4: 某燈泡的電流及電阻分別為0.25A及880Ω，計算電壓及電功率。

按 Prog 1 再按 EXE (不輸入電壓) 0.25 EXE 880 EXE

EXE (不輸入電功率，顯示半電壓為220 V)

EXE (顯示電功率為 55 W)

　

例題5: 某燈泡的電流及電功率分別為0.25A及55W，計算電壓及電阻。

按 Prog 1 再按 EXE (不輸入電壓) 0.25 EXE EXE (不輸入電阻)

55 EXE (顯示電壓為 220 V)

EXE (顯示電阻為 880 Ω)

計算完結後按 AC 終止程式

　

例題5: 某燈泡的電阻及電功率分別為880Ω及55W，計算電壓及電流。

按 Prog 1 再按 EXE EXE (不輸入電壓及電流)

880 EXE 55 EXE (顯示電壓為 220 V)

EXE (顯示電流為 0.25 A)

計算完結後按 AC 終止程式

　

註: 程式執行完結後按 RCL A、RCL B、RCL C及RCL D可顯示電壓、電流、電阻及電功率的數值。
