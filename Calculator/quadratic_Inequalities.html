一元二次不等式

Quadratic inequalities in one variable

這個程式根本網友梁立江提供fx-3650P程式修改至適合fx-50FH使用， 程式可以解一元二次不等式，程式亦可以用作計算一元二次方程。

網友提供程式日期: 2012年12月29日

程式 (70 bytes)

ClrMemory: ?→X: ?→A: ?→B: ?→C: X => A>0M+:

X=0 => 0>AM+ : √(B2 - 4AC) cos( πr(A>0 :

(Ans  - B) ┘(2A→X◢ - B┘A - Ans→Y

　

註1: 程式會由小至大產生一元二次方程式的兩個根(α及β)，在解一元二次不等式時，若果顯示答案時沒有細字M，表示解為 α < x < β 或 α ≦ x ≦ β，若果有顯示細字M，則表示解為 (x < α 或 x > β) 或 (x ≦α 或 x ≧ β)。

註2: 若出現Math ERROR，計算一元二次方程時代表無解，若是計算一元二次不等式則可能是無解或無限解。

　

例題1: 解 x2 - 7x + 12 < 0

按 Prog 1 再按 EXE (不輸入或輸入0代表 < 0 或 ≦ 0)

1 EXE - 7 EXE 12 EXE (沒有M字及顯示3) EXE (沒有M字及顯示4)

所以解為 3 < x < 4

　

例題2: 解 x2 - 7x + 12 ≧ 0

按 Prog 1 再按 1 EXE (輸入1代表 > 0 或 ≧ 0)

1 EXE - 7 EXE 12 EXE (有M字及顯示3) EXE (有M字及顯示4)

所以解為 x ≦3 或 x ≧ 4

　

例題3: 解 x2 - 7x + 12 = 0

按 Prog 1 再按 EXE (計算二次方程時，可以不輸入數值，實際輸入任何數值都可以)

1 EXE - 7 EXE 12 EXE (顯示3) EXE (顯示4)

所以解為 x = 3 或 x = 4
