第一類貝塞爾函數(Bessel Function of the first kind)

更新日期: 2014年8月1日

程式可以計第一類貝塞爾函數 Jn(x) (Bessel Function of the first kind)，其中 n 為整數。

注意: πr 是按 SHIFT EXP SHIFT Ans 2

程式 (59 bytes)

ClrMemory: ?→A: ?→X: Lbl 0: M→C:

(X÷2)^( 2B + A) cos( πrB ) ÷ ( B! (B+A)! M+:

B+1→B: M-C => Goto 0: M

註: 如果是使用fx-3650P II或fx-50FH II，上述程式中的cos( πrB ) ÷ ( B! (B+A)! M+:可以簡化為 cos( πrB ) ÷ B! (B+A)! M+: ，可節省1 byte。

　

計算第一類貝塞爾函數 J3(1.3)的數值。

按 Prog 1  再按 3 EXE 1.3 EXE (顯示答案為0.041135825)
