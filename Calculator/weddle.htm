Weddle's Rule 定積分計算

更新日期: 2014年8月31日

程式 (最少96 bytes)

ClrMemory: ?→M: ?→Y: ?→A: (Y - M) ÷ ( 6A→D:

Lbl 0: ln( M: B + Ans(1 + 4(C2+5=6C) + 5(C=3→B:

DM+: 1+C→C: C=7 => DM- => 1+X→X: Ans => 0→C:

A>X => Goto 0: 3BD÷10

　

註1: 程式要輸入的區間數目n是指將定積分計算的範圍平分為n個區間，而每一個區間都會應用Weddle's rule計算，即是將該區間再分六分計算七個對應函數值，然後再應用法則計算定積分。

註2: 綠色的 ln( M是函數方程(變數是X)，若果想計算其它函數的積分，只要修改綠色的部份。

註3: 如果是使用fx-3650P II或fx-50FH II，上述程式中紫色的開括號 ( 可以省略，程式長度可減1 byte。
