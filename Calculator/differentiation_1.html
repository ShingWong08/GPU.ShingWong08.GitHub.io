微分計算(II)

程式編寫日期: 2008年2月1日

程式可計算一個函數的第一、第二、第三及第四導數的近似值。不過要留意第四導數的準確會較低。

注意: πr 是按 Shift EXP Shift Ans 2

程式 (最少122 bytes)

ClrMemory: ?→X: Lbl 0: X + ln( X →Y: M=0 => Y→A◢

M=1 => Y→B: M=2 => Y→C: M=3 => Y→D: 1M+:

M cos( πrM ) ÷ 128 + X→X: M≠5 => Goto 0: 64(C - B◢

4^( 7 )(C - 2A + B◢ 4^( 10 )(Y - 2C + 2B - D◢

4^( 14 )(Y - 4C + 6A - 4B + D

註: 綠色的X + ln( X是函數方程(變數是X)，若果想計算其它方程，只要修改綠色的部份。

　

例題: 計算 f(x) = x + ln x 在 x = 2的第一、第二、第三及第四導數的近似值。

按 Prog 1 再按 2 EXE (顯示 f(2)為 2.693147181)

EXE (顯示f'(2)為1.500002543)

EXE (顯示f''(2)為 -0.250001907)

EXE (顯示f (3) (2)為 0.250011451)

EXE (顯示f (4) (2)為 -0.375025806)
